This is the Assignment2 for iPad and iPhone development by Stanford University (CSE 193p http://www.stanford.edu/class/cs193p/cgi-bin/drupal/) . The product of this assignment is a programmable calculator. Detailed information can be found in the pdf file in root directory

Asst2 implemented a programmable calculator, the a,b,c buttons denotes three variables, the value of the variables can be changed by pressing Test1,Test2 or Test3. Also this asst implemented a new undo button, which would undo the previous action.

Demo, Press Test1 , 3 ,Enter , a, + would return the value of 3+7 = 10
